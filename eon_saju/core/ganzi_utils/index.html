간지(干支) 동적 계산 유틸리티

월간지, 일간지, 시간지를 동적으로 계산하는 공통 함수를 제공합니다. FourPillars::calculate_* 함수들과 동일한 로직을 사용하여 정합성을 보장합니다.

§용도

  • Fuzzer에서 고해상도 타임라인 조사 시 사용
  • 주어진 날짜/시간의 간지를 빠르게 계산할 때 사용
