# Eon - 운명 플랫폼

다양한 운명 체계를 통합하는 AI 기반 운명 분석 플랫폼입니다.

## 지원 시스템

| 시스템 | 핵심 입력값 | 주요 로직 | 특징 |
|--------|------------|----------|------|
| **사주** | 생년월일시 | 오행의 생극제화 | 시간의 순환성 강조 |
| 베딕 | 행성 정밀 좌표 | 나크샤트라, 다샤 | 천문학적 정확도 중시 |
| 자미두수 | 생년월일시 | 별들의 궁 배치 | 매우 세밀한 데이터 포인트 |
| 주역 | 현재의 선택/시점 | 64괘 이진 논리 | 변화의 패턴 분석 |
| 휴먼 디자인 | 생년월일시 | 점성술 + 유전학 결합 | 하이브리드 아키텍처 |

## 크레이트 구조

```
crates/
├── eon-core/     # 공통 타입 및 유틸리티
├── eon-saju/     # 사주 계산 로직
└── eon-data/     # 만세력 등 정적 데이터
```

## 빌드

```bash
cargo build --workspace
```

## 테스트
